You are an expert UX/UI designer.

Design a modern, professional, and visually rich **agriculture marketplace platform UI called "AGRI GENIX"** in Figma.

The platform connects **Farmers, Buyers, and Godown (Warehouse) Owners** in one ecosystem.

The design should be **farmer-friendly, simple, and visually clear**, but also look like a **modern startup product**.

Use a **green agriculture theme with natural colors**, modern cards, soft shadows, rounded corners, and large icons.

The UI must be **mobile-first and responsive** because most farmers will use smartphones.

---

DESIGN STYLE

• Clean modern UI
• Agriculture inspired color palette (green, light brown, white)
• Large buttons and icons for easy usability
• Minimal text for farmers with low digital literacy
• Friendly illustrations (farmers, crops, storage warehouses)
• Rounded cards and soft shadows
• Clear hierarchy and spacing

---

USER TYPES

The platform must support 3 user roles:

1. Farmers
2. Buyers (Traders / Retailers / Companies)
3. Godown Owners (Warehouse providers)

Each role must have a **separate dashboard and features**.
